2. Verification status and verification date must be separate from the license label.
3. English translations require faithful meaning and should not broaden source claims.
4. Category changes must avoid silent fragmentation of the catalog vocabulary.
5. Dead links, changed terms, and unavailable downloads should be recorded, not hidden.
6. No real personal information should be added as sample catalog content.
